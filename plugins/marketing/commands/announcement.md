# Announcement Generator

Create announcement copy for various platforms and purposes.

## Process

1. Understand the announcement type and target
2. Generate platform-appropriate content
3. Include relevant assets/links

## Output Format

```markdown
## Announcement: [Type] - [Subject]

### Twitter/X Post (280 chars)
[Concise, punchy, with relevant emoji and hashtags]

### Discord/Community Post
```
[Subject Line/Ping]

[Opening hook]

[Main content - 2-3 paragraphs]

[Key details/links]

[Call to action]
```

### Press Release Format (if applicable)
```
FOR IMMEDIATE RELEASE

[HEADLINE]

[SUBHEADLINE]

[CITY, DATE] - [Opening paragraph - who, what, when, why]

[Details paragraph]

[Quote from developer]

[Features/bullet points]

[Availability and pricing]

[About section]

[Contact info]
```

### Blog/Long Form
[Full announcement with proper structure]

### Visual Assets Needed
- [ ] [Asset 1 with specs]
- [ ] [Asset 2 with specs]

### Timing Recommendations
- Best post time: [Time/Day]
- Embargo considerations: [If applicable]

### Follow-up Plan
- Response to comments: [Approach]
- Follow-up posts: [Planned content]
```

## Announcement Types

**Game Reveal:** Maximum excitement, mystery
**Release Date:** Clarity, hype, wishlist CTA
**Update/Patch:** What's new, community thanks
**Early Access:** Set expectations, roadmap
**Sale:** Urgency, value proposition
**Milestone:** Celebration, community thanks

Generate based on user's announcement needs.
