# Trailer Planner

## Core Philosophy

```markdown

**Goal:** Stop the scroll

### Core Loop (3-15s)

3. [Action/moment] - [Duration]

**Goal:** Show depth and breadth
**Shots:**
1. [Feature/mode] - [Duration]
2. [Feature/mode] - [Duration]
3. [Feature/mode] - [Duration]

### Close (30-45s)
**Goal:** Brand and call to action
**Elements:**
- Game title
- Platform icons
- Release date or "Wishlist Now"
- Developer logo (brief)
```
