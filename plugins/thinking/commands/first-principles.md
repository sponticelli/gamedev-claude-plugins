# First Principles Thinking

Strip away assumptions to find fundamental truths, then build solutions from the ground up.

## Why It Works

Most thinking is by analogy: "This is like that, so do what worked before." First principles asks: "What's actually true here?" This is how breakthrough innovation happens.

## Process

1. Identify the problem and conventional approach
2. List all assumptions embedded in that approach
3. Question each assumption ruthlessly
4. Identify what's fundamentally, unavoidably true
5. Build a new solution from those fundamentals

## Example: Elon Musk on Batteries

**Conventional:** "Batteries cost $600/kWh, that's just what they cost."

- What are batteries made of? (Cobalt, nickel, aluminum, carbon, polymers, steel)
- What do those materials cost on the London Metal Exchange? (~$80/kWh)

## Questioning Prompts

- "Why does it have to be this way?"
- "What if we started from scratch?"
- "What's the physics/logic here?"
- "Who says this is a rule?"
- "What would a Martian think about this?"

Apply to user's problem immediately.
