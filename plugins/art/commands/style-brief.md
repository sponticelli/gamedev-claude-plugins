# Art Style Brief Generator

Create a rapid art style brief from a game concept.

## Output Format

```markdown
# Art Style Brief: [Game Title]

## Visual Identity in One Sentence
[Capture the entire visual approach in one sentence]

## Style Keywords
[5 adjectives that describe the visual style]

## Color Direction
### Primary Palette
- **Dominant:** [Color] - [What it represents]
- **Accent:** [Color] - [When used]
- **Neutral:** [Color] - [Base tones]

### Mood Mapping
| Mood | Colors |
|------|--------|
| [Mood 1] | [Colors] |
| [Mood 2] | [Colors] |

## Shape Language
- **Heroes/Allies:** [Shapes] - [Why]
- **Threats/Enemies:** [Shapes] - [Why]
- **Environment:** [Shapes] - [Why]

## Reference Touchstones
| Reference | What to Take |
|-----------|-------------|
| [Game/Film/Art] | [Specific element] |
| [Game/Film/Art] | [Specific element] |
| [Game/Film/Art] | [Specific element] |

## What We're NOT
[Styles or approaches to explicitly avoid]

## Production Considerations
- **Complexity:** [Low/Medium/High]
- **Team Size:** [Solo/Small/Large]
- **Key Challenge:** [Main production concern]

## First Art Test
[What single image would prove/disprove this style works]
```

Generate based on user's game concept.
