# Juice Audit

Quick analysis of a specific game moment, mechanic, or feature for missing juice.

## Process

1. Identify the feature/moment to audit
2. Check all four juice pillars
3. Identify biggest gaps
4. Prescribe quick fixes
5. Suggest reference games

## Quick Diagnostic Questions

If the user hasn't described the issue, ask:
1. "What action or moment feels flat?"
2. "What happens when the player does [X]?"
3. "What sounds play during this?"
4. "What's the current animation like?"
